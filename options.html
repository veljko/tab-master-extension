Keyboard Shortcuts

MRU Quick Switch Alt+W
MRU Normal Switch (forward) Unassigned
MRU Normal Switch (backward) Unassigned
Move tab left Shift+Alt+←
Move tab right Shift+Alt+→
Move tab to first position Shift+Alt+↑
Move tab to last position Unassigned

All shortcuts are customizable via chrome://extensions/shortcuts (or edge://extensions/shortcuts in Edge).
If you want to use Ctrl+Tab for MRU switching, please use the following AutoHotkey script as a workaround: 
#Requires AutoHotkey v2.0
#SingleInstance Force

; Remap Ctrl+Tab -> Alt+W for Tab Master extension (MRU quick switch)
; Scoped to Edge and Chrome only - doesn't affect other apps.

#HotIf WinActive("ahk_exe msedge.exe")
^Tab:: Send("!w")
^+Tab:: Send("!w")   ; Ctrl+Shift+Tab also triggers quick switch (keeps cycling)
#HotIf

#HotIf WinActive("ahk_exe chrome.exe")
^Tab:: Send("!w")
^+Tab:: Send("!w")
#HotIf
